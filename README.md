A scalable, highly available, and secure web application hosted on AWS, following the AWS Well-Architected Framework principles.

## Components
- Amazon EC2 (Ubuntu)
- Application Load Balancer
- Amazon RDS (MySQL)
- AWS Secrets Manager
- Auto Scaling Group
- VPC with Public and Private Subnets
